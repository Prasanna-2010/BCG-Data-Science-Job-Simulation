## 📌 Project Overview

This repository contains my solution for the **BCG Data Science Job Simulation** offered by **Forage**.

The project focuses on solving a real-world business problem by predicting **customer churn** using machine learning techniques. The workflow covers the complete data science lifecycle—from data exploration and feature engineering to model development and business recommendations.

---

## 🎯 Business Objective

Customer churn is one of the biggest challenges faced by subscription-based businesses.

The objective of this project is to:

- Analyze customer and pricing data
- Identify factors contributing to customer churn
- Build a predictive machine learning model
- Generate business insights to improve customer retention

# 📈 Machine Learning Model

**Algorithm Used**

- Random Forest Classifier

The model was trained to predict customer churn based on customer usage patterns, pricing history, and engineered features.

Model evaluation included:

- Accuracy
- Classification Report
- Confusion Matrix
- Feature Importance

---

# 📊 Key Insights

- Customer churn is influenced by multiple customer behavior factors.
- Feature engineering improved the predictive capability of the model.
- Historical pricing alone is not the strongest indicator of churn.
- Customer usage and contract-related features significantly contribute to churn prediction.
- Machine learning enables proactive identification of customers likely to churn.
